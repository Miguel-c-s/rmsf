***ESP32 code FOR PROJECT WITH APP-SERVER-MICROCONTROLLER CONNECTION***

Requirements:
- Having Arduino IDE installed
- Having ESP32 library installed (https://github.com/espressif/arduino-esp32)
- Change ESP32 partitions sizes (https://desire.giesecke.tk/index.php/2018/01/30/change-partition-size/)

Usage:
- Open the file with Arduino IDE
- Choose the correct serial port
- compile the code and program the board

IMPORTANT NOTES:
- change MQTT server
- change WIFI settings
